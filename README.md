# 🌌Milky-Way-Space-Engine

What if you could hold the universe inside your browser?

This project started with a simple idea — not just creating planets, but creating the feeling of floating through space. Inspired by NASA visualizations, sci-fi movies, and futuristic web experiences, this interactive Solar System transforms code into a cinematic journey through the cosmos.

As glowing planets orbit around the Sun and stars stretch endlessly across deep space, the experience creates the illusion of traveling through a living universe. Built using modern 3D web technologies, the project combines creativity, real-time rendering, and immersive interaction into one futuristic simulation.

## ⚙️ RUN LOCALLY

Clone the repository:

```bash id="7xvmlz"
git clone https://github.com/augsatr/3D-Solar-System.git
```

Go inside the project:

```bash id="shm31k"
cd 3D-Solar-System
```

Install dependencies:

```bash id="8jv5mp"
npm install
```

Start the development server:

```bash id="j7px9a"
npm run dev
```

# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](https://oxc.rs)
- [@vitejs/plugin-react-swc](https://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](https://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](https://react.dev/learn/react-compiler/installation).
